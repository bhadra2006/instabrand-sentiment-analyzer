# instabrand-sentiment-analyzer
A Python-based Instagram Comment Sentiment Analyzer that automatically extracts comments from Instagram posts using Playwright and performs multilingual sentiment analysis with Hugging Face Transformers, presented through an interactive Streamlit dashboard.

---

## Features

- Automatic Instagram comment extraction
- User-defined number of posts to analyze
- Duplicate comment removal
- Multilingual sentiment analysis
- Positive, Negative and Neutral classification
- Overall customer opinion
- Professional Streamlit dashboard
- Automatic report generation

---

## Tech Stack

- Python
- Playwright
- Streamlit
- Pandas
- Hugging Face Transformers
- PyTorch

---

## Workflow

```
Instagram Username
        │
        ▼
Fetch Comments using Playwright
        │
        ▼
comments.csv
        │
        ▼
Sentiment Analysis
        │
        ▼
Generate Report
        │
        ▼
Display Dashboard
```
